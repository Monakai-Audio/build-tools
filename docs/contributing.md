# Contributing to Build Tools

Thank you for your interest in improving Monakai Audio's open-core build
tooling.

## How to contribute

1. Fork the repository and create a feature branch.
2. Make focused changes with clear commit messages.
3. Ensure the CI workflow passes on Windows and Ubuntu.
4. Open a pull request describing the change and the problem it solves.

## Scope

This repository is for build scripts, templates, and documentation only. Do not
add proprietary DSP algorithms, preset banks, synthesis code, or brand assets.

## Code style

- CMake: lowercase commands, 4-space indentation.
- PowerShell: follow the comment-based help pattern in existing scripts.
- Documentation: keep guides concise and platform-specific.
